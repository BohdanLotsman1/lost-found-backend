## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Knex

```bash
# migrate create
$ knex seed:make {migrate_name}

# migrate run
$ knex migrate:latest

# migrate rollback
$ knex migrate:rollback

# seed create
$ knex seed:make {seed_name}

# seed run
$ knex seed:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
